._drawDirty()

src/graphics/canvas-layer.js#L189-L203
public ._drawDirty()
  • Triggered by the "RenderScene" event
  • If the number of rects is over 60% of the total number of objects do the naive method redrawing CanvasLayer.drawAll instead
  • Otherwise, clear the dirty regions, and redraw entities overlapping the dirty regions.
